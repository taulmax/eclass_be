# 웹프로그래밍 미니프로젝트!

## Get started

React의 build 파일을 이미 넣어놓아서 프론트단에서는 할게 없습니다.
Express에서만 install이 필요합니다.

    npm install // node_modules 설치
    nodemon server.js // 서버 실행

React 웹서버를 키거나 코드 수정사항을 반영하고 싶으면 `eclass_fe` 폴더 안에서도 설치를 진행해주시면 됩니다.

    npm install // node_modules 설치
    npm start // 웹서버 실행
    npm build // 빌드 파일 생성
